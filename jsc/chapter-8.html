(গ) প্রমান কর যে প্রদত্ত রাশির বর্গ ও (a + c)2 - b2 সমান নয়।

    সমাধান :

    এখন, প্রদত্ত রাশির বর্গ= (a - b + c)2

    ={(a - b) + (c)}2

   = (a - b)2 + 2.(a - b)(c) + c2

    = a2 - 2ab + b2 + 2ac - 2bc + c2

    = a2 + b2 + c2 - 2ab - 2bc + 2ac

    এবং (a + c)2 - b2 = a2 + 2ac + c2 - b2

    = a2 - b2 + c2 + 2ac

    দেখা যাচ্ছে (a - b + c) এর বর্গ এবং (a + c)2 - b2 এর  মান একই নয়।

    সুতরাং প্রদত্ত রাশির বর্গ ও (a + c)2 - b2 সমান নয় (প্রমাণিত)
