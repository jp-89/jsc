৪.১.২(ক) সরল কর:
$(x+y)^2+2(x+y)(x-y )+(x-y)^2$
সমাধান:
প্রদত্ত রাশী
= $(x+y)^2+2(x+y)(x-y )+(x-y)^2$

ধরি,
$ x+y =a $ এবং
$ x+y = b $

সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^2+2ab+b^2$
=$(a+b)^2 $

এখন, মান বসিয়ে পাই,
=$(x+y+x-y)^2 $
=$(2x)^2 $
= $4x^2 $

অতএব,
নির্ণেয় মান: = $4x^2 $

৪.১.২(খ) সরল কর:
$(2a+3b)^2-2(2a+3b)(3b-a )+(3b-a)^2$
সমাধান:
প্রদত্ত রাশী
= $(2a+3b)^2-2(2a+3b)(3b-a )+(3b-a)^2$

ধরি,
$ 2a+3b = x $ এবং
$ 3b-a = y $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $x^2-2xy+y^2$
=$(x-y)^2 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $\{2a+3b-(3b-a)\}^2 $
= $(2a+3b-3b+a)^2 $
=$(3a)^2 $
= $9a^2 $

অতএব,
নির্ণেয় মান: = $9a^2 $

৪.১.২(গ) সরল কর:
$(3x^2+7y^2)^2+2(3x^2+7y^2)(3x^2-7y^2 )+(3x^2-7y^2)^2$
সমাধান:
প্রদত্ত রাশী
= $(3x^2+7y^2)^2+2(3x^2+7y^2)(3x^2-7y^2 )+(3x^2-7y^2)^2$

ধরি,
$ 3x^2+7y^2 = a $ এবং
$ 3x^2+7y^2 = b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^2+2ab+b^2$
=$(a+b)^2 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $(3x^2+7y^2+3x^2-7y^2)^2 $
= $(6x^2)^2 $
= $36x^4 $

অতএব,
নির্ণেয় মান: = $36x^4 $

৪.১.২(ঘ) সরল কর:
$(8x+y)^2-(16x+2y)(5x+y )+(5x+y)^2$
সমাধান:
প্রদত্ত রাশী
= $(8x+y)^2-(16x+2y)(5x+y )+(5x+y)^2$
= $(8x+y)^2-2(8x+y)(5x+y )+(5x+y)^2$

ধরি,
$ 8x+y= a $ এবং
$ 5x+y = b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^2-2ab+b^2$
=$(a-b)^2 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $\{(8x+y)-( 5x+y)\}^2 $
= $(8x+y-5x-y)^2 $
= $(3x)^2 $
= $9x^2 $

অতএব,
নির্ণেয় মান: = $9x^2 $

৪.১.২(ঙ) সরল কর:
$(5x^2-3x-2)^2 + (2+5x^2-3x)^2-2(5x^2-3x-2 )(2+5x^2-3x)$
সমাধান:
প্রদত্ত রাশী
= $(5x^2-3x-2)^2 + (2+5x^2-3x)^2-2(5x^2-3x-2 )(2+5x^2-3x)$
= $(5x^2-3x-2)^2 -2(5x^2-3x-2 )(2+5x^2-3x)+ + (2+5x^2-3x)^2$

ধরি,
$ 5x^2-3x-2= a $ এবং
$ 2+5x^2-3x= b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^2-2ab+b^2$
=$(a-b)^2 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $\{(5x^2-3x-2)-( 2+5x^2-3x)\}^2 $
= $(5x^2-3x-2-2-5x^2+3x)^2 $
= $(-4)^2 $
= $16 $

অতএব,
নির্ণেয় মান: = $16 $
