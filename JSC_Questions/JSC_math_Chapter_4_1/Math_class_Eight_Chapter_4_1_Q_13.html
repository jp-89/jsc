৪.১.১৩। দুইটি রাশির বর্গের অন্তররূপে প্রকাশ কর:
(ক) $(5p-3q)(p+7q) $
সমাধান:
প্রদত্ত রাশী = $(5p-3q)(p+7q) $

আমরা জানি, $ ab = ({a+b}/2)^2 - ({a-b}/2)^2$

ধরি,
$ 5p-3q =a $ এবং
$ p+7q = b $

সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $\{ {( 5p-3q)+ (p+7q)} /2 \}^2 - \{ { ( 5p-3q)- (p+7q)} /2 \}^2$

= $\{ {( 5p-3q+p+7q)} /2 \}^2 - \{ { ( 5p-3q- p-7q)} /2 \}^2$

= $\{ {( 6p+4q)} /2 \}^2 - \{ { ( 4p-10q)} /2 \}^2$

= $( 3p+2q)^2 - ( 2p-5q)^2$


৪.১.১৩। দুইটি রাশির বর্গের অন্তররূপে প্রকাশ কর:
(খ) $6a+9b)(7b-8a) $
সমাধান:
প্রদত্ত রাশী = $6a+9b)(7b-8a) $

আমরা জানি, $ xy = ({x+y}/2)^2 - ({x-y}/2)^2$

ধরি,
$ 6a+9b =x $ এবং
$ 7b-8a = y $

সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $\{ {( 6a+9b)+ (7b-8a)} /2 \}^2 - \{ { ( 6a+9b)- (7b-8a)} /2 \}^2$

= $\{ {( 6a+9b+7b-8a)} /2 \}^2 - \{ { ( 6a+9b- 7b+8a)} /2 \}^2$

= $\{ {( -2a+16b)} /2 \}^2 - \{ { ( 14a-2b)} /2 \}^2$

= $\{ {( 16b-2a)} /2 \}^2 - \{ { -( 2b-14a)} /2 \}^2$

= $\{ {( 8b-a)} \}^2 - \{ { ( b-7a)} \}^2$

= $( 8b-a)^2 - ( b-7a)^2$


৪.১.১৩। দুইটি রাশির বর্গের অন্তররূপে প্রকাশ কর:

প্রদত্ত রাশী = $(3x+5y)(7x-5y) $

ধরি,
$3x+5y =a $ এবং
$ 7x-5y= b $

সুতরাং, আমরা পাই,
