২.১.১০। শতকরা বার্ষিক যে হারে কোনো মূলধন ৬ বছরে মুনাফা-মূলধনে দ্বিগুন হয়, সেই হারে কত টাকা ৪ বছরে মুনাফা মূলধনে ২০৫০ টাকা হবে?
সমাধান:
মনে করি, মূলধন = ১০০ টাকা।
প্রশ্নমতে, ৬ বছরে মুনাফা-মূলধন = ১০০ ×২ = ২০০ টাকা।

অতএব, ৬ বছরে মুনাফা = (২০০-১০০ ) টাকা।
= ১০০ টাকা।

অতএব,
১০০ টাকার ৬ বছরের মুনাফা ১০০ টাকা।
১০০ টাকার ১ বছরের মুনাফা ${১০০}/৪ $ টাকা।

১০০ টাকার ৪ বছরের মুনাফা ${ ১০০× ৪}/৬ $ টাকা।

= ${২০০}/৩ $ টাকা।

সুতরাং,
১০০ টাকার ৪ বছরে মুনাফাসহ মূলধন = $১০০+ {২০০}/ ৩ $ টাকা।

=${ ৩০০+২০০}/৩ $

= ${৫০০}/৩ $ টাকা।

প্রশ্নানুসারে,
মুনাফাসহ মূলধন ${৫০০}/৩ $ হলে মূলধন ১০০ টাকা।

মুনাফাসহ মূলধন ১ হলে মূলধন ${১০০}/ {{৫০০}/৩} $ টাকা।

= ${১০০ ×৩ }/{ ৫০০} $ টাকা

মুনাফাসহ মূলধন ২০৫০ হলে মূলধন $ {১০০ ×৩ ×২০৫০}/ {৫০০} $ টাকা।

= ১২৩০ টাকা।

নির্ণেয়, মূলধন = ১২৩০ টাকা।

বিকল্প পদ্ধতি:

আসল = P
মুনাফা = I = 2P -P = P
মুনাফার হার = r
সময় = n = ৬ বছর।

সুতরাং, মান বসিয়ে পাই,
$ P = P× r/{১০০} ×৬ $

বা, $ P = {P× r ×৬}/{১০০} $ ...(1)

দ্বিতীয় ক্ষেত্রে,

মুনাফা = I = ২০৫০-P
মুনাফার হার = r
সময় = n = ৪ বছর।

সুতরাং, মান বসিয়ে পাই,
$ ২০৫০-P = P× r/{১০০} ×৪ $

বা, $ ২০৫০-P = { P× r ×৪}/{১০০} $ ...(2)

(2) সমীকরনকে (1) দ্বারা ভাগ করিয়া পাই,
${ ২০৫০-P}/{ P} = {{ P× r ×৪}/{১০০}} / { {P× r ×৬}/{১০০} } $

বা, $(২০৫০-P)×৬ = ৪×P $

বা, $১২৩০০ - ৬P = ৪P $

বা, $১০P = ১২৩০০$

বা, $P = {১২৩০০}/{১০}$
