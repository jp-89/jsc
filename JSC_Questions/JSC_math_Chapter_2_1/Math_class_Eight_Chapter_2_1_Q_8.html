২.১.৮। ৬৫০০ টাকা যে হার মুনাফায় ৪ বছরে মুনাফা-আসলে ৮৮৪০ টাকা হয়, ঐ একই হার মুনাফায় কত টাকা ৪ বছরে মুনাফা-আসলে ১০২০০ টাকা হবে।
সমাধান:
আসল = ৬৫০০ টাকা।
৪ বছরে মুনাফা সহ আসল = $ ৮৮৪০$ টাকা।
সুতরাং,
৪ বছরে মুনাফা = $(৮৮৪০ - ৬৫০০) $ টাকা।
= $২৩৪০$ টাকা।

অতএব,
১ বছরে মুনাফা = ${২৩৪০}/ ৪ $ টাকা।
= $৫৮৫$ টাকা।

সুতরাং,
৬৫০০ টাকার ১ বছরের মুনাফা = ৫৮৫ টাকা।
১ টাকার ১ বছরের মুনাফা = ${৫৮৫}/{৬৫০০}$ টাকা।

১০০ টাকার ১ বছরের মুনাফা = ${৫৮৫ ×১০০}/{৬৫০০}$ টাকা।

= $৯$ টাকা।

অতএব, মুনাফার শতকরা হার = $৯% $

সুতরাং,
১০০ টাকার ১ বছরের মুনাফা ৯ টাকা।
১০০ টাকার ৪ বছরের মুনাফা $ ৯ × ৪ $ টাকা। = ৩৬ টাকা।

অতএব, মুনাফা সহ আসল = ১০০+৩৬ = ১৩৬ টাকা।

অতএব,
মুনাফা-আসলে ১৩৬ টাকা হলে আসল = ১০০ টাকা।
মুনাফা-আসলে ১ টাকা হলে আসল =${১০০}/{ ১৩৬}$ টাকা।

মুনাফা-আসলে ১০২০০ টাকা হলে আসল = ${১০০× ১০২০০}/{ ১৩৬}$ টাকা।
= ৭৫০০ টাকা।

নির্ণেয়, আসল= ৭৫০০ টাকা।

বিকল্প পদ্ধতি:
আমরা জানি, সরল মুনাফার ক্ষেত্রে,
I = Prn
বা, $r = {I}/{Pn } $
অর্থাৎ,
$ \text " মুনাফার হার " ={ \text " মুনাফা " }/{ \text" আসল " × \text" সময় " } $

এখানে, প্রথম ক্ষেত্রে
আসল = ৬৫০০ টাকা।
৪ বছরে মুনাফা সহ আসল = $ ৮৮৪০$ টাকা।
সুতরাং,
৪ বছরে মুনাফা = $(৮৮৪০ - ৬৫০০) $ টাকা।
= $২৩৪০$ টাকা।

মুনাফা = I = ২৩৪০টাকা।
আসল = P = ৬৫০০ টাকা।
মুনাফার হার = r = ?
সময় = n = ৪ বছর।

এখন, মান বসিয়ে পাই,
$ \text " মুনাফার হার " ={ ২৩৪০ }/{ ৬৫০০ × ৪ } $

= $০.০৯ $
অতএব, মুনাফার শতকরা হার = $০.০৯ × ১০০ $
=$ ৯ % $

দ্বিতীয় ক্ষেত্রে
আমরা জানি, সরল মুনাফার ক্ষেত্রে,
A = P(১+rn)
বা, $ P = A/{১+rn}$

এখানে,
মুনাফা = I = ২৩৪০টাকা।
মুনাফা-আসল = A = ১০২০০ টাকা।
আসল = P = ? টাকা।
মুনাফার হার = r = ০.০৯
সময় = n = ৪ বছর।

এখন, মান বসিয়ে পাই,
$ P = {১০২০০}/{1+০.০৯×৪}$

$ = {১০২০০}/{১+০.৩৬ }$

$ = {১০২০০}/{১.৩৬ }$

$ = ৭৫০০ $

নির্ণেয়, আসল= ৭৫০০ টাকা।
