২.১.-সৃজনশীল- ৩। জলিল তার মোবাইলটি ৪০০০ তাকে বিক্রি করলেন। তিনি হিসাব করে দেখলেন যে, তার ১০% ক্ষতি হয়েছে।

ক) জলিলের কত টাকা ক্ষতি হয়েছে?
খ) জলিল যদি ১০% লাভ করতে চায় তবে তাকে কত তাকে মোবাইলটি বিক্রি করতে হবে?
গ) ক্রয়মূল্যের সমান টাকা ৮% মুনাফায় ব্যাংকে জমা রাখলে ৫ বছর পর মুনাফা-আসলে তিনি কত টাকা পাবেন?
সৃজনশীল ৩ (ক) জলিলের কত টাকা ক্ষতি হয়েছে?
সমাধান:
১০% ক্ষতিতে বিক্রয়মূল্য (১০০-১০) টাকা
= ৯০ টাকা।

অতএব,
বিক্রয়মূল্য ৯০ টাকা হলে ক্রয়মূল্য ১০০ টাকা।
বিক্রয়মূল্য ১ টাকা হলে ক্রয়মূল্য $ {১০০} /{৯০ } $ টাকা।
বিক্রয়মূল্য ৪০০০ টাকা হলে ক্রয়মূল্য ${ ১০০ ×৪০০০ }/{৯০ } $ টাকা।
৪৪৪.৪৪৪ টাকা।

অতএব,
জলিলের ক্ষতি হয়েছে = ৪৪৪৪.৪৪-৪০০০ টাকা
= ৪৪৪.৪৪৪ টাকা।

সৃজনশীল ৩ (খ) জলিল যদি ১০% লাভ করতে চায় তবে তাকে কত তাকে মোবাইলটি বিক্রি করতে হবে?
 সমাধান:
১০% লাভে বিক্রয়মূল্য (১০০+১০) টাকা
= ১১০ টাকা।
এখানে, মোবাইটির ক্রয়মূল্য ৪৪৪৪.৪৪ টাকা।

অতএব,
ক্রয়মূল্য ১০০ টাকা হলে বিক্রয়মূল্য ১১০ টাকা।
ক্রয়মূল্য ১ টাকা হলে বিক্রয়মূল্য $ {১১০} /{ ১০০ } $ টাকা।
ক্রয়মূল্য ৪৪৪৪.৪৪ টাকা হলে বিক্রয়মূল্য ${ ১১০ ×৪৪৪৪.৪৪ }/{ ১০০ } $ টাকা।
৪৮৮৮.৮৮ টাকা।

অতএব,
১০% লাভ বিক্রয় করতে হলে ৪৮৮৮.৮৮ টাকায় বিক্রি করতে হবে।

সৃজনশীল ৩ (গ) ক্রয়মূল্যের সমান টাকা ৮% মুনাফায় ব্যাংকে জমা রাখলে ৫ বছর পর মুনাফা-আসলে তিনি কত টাকা পাবেন?
সমাধান:
দেওয়া আছে,
'ক' থেকে প্রাপ্ত আসল = ৪৪৪৪.৪৪ টাকা।
আমরা জানি, সরল মুনাফার ক্ষেত্রে,
I = Prn

এখানে,
আসল P = ৪৪৪৪.৪৪ টাকা।
মুনাফা I = ?
মুনাফার হার r = ৮% = $ ৮/{ ১০০ } $
সময় n = ৫ বছর

সুতরাং, মান বসিয়ে পাই,
$I = ৪৪৪৪.৪৪×{ ৮} /{ ১০০ } ×৫ $

বা, $I = {৪৪৪৪.৪৪× ৮ }/{ ১০০ } ×৫ $

বা, $ I = ১৭৭৭.৭৭ $

৫ বছর পর মুনাফা-আসল = ৪৪৪৪.৪৪+১৭৭৭.৭৭ টাকা।
৬২২২.২১ টাকা।

অতএব,
নিণেয় তিনি ৫ বছর পর ৬২২২.২১ টাকা পাবেন।
