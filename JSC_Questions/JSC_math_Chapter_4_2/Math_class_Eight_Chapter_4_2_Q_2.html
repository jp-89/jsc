৪.২.২(ক) সরল কর:
$(3x+y)^3 +3(3x+y)^2(3x-y)+3(3x+y)(3x-y)^2+ (3x-y)^3$
সমাধান:
প্রদত্ত রাশী
= $(3x+y)^3 +3(3x+y)^2(3x-y)+3(3x+y)(3x-y)^2+ (3x-y)^3$

ধরি,
$ 3x+y =a $ এবং
$ 3x-y = b $

সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^3+3a^2b+3ab^2+b^3$
=$(a+b)^3 $

এখন, মান বসিয়ে পাই,
=$(3x+y+3x-y)^3 $
=$(6x)^3 $
= $216x^3 $

অতএব,
নির্ণেয় মান: = $216x^3 $

৪.২.২(খ) সরল কর:
$(2p+5q)^3 +3(2p+5q)^2(5q-2p)+3(2p+5q)(5q-2p)^2+ (5q-2p)^3$
সমাধান:
প্রদত্ত রাশী
= $(2p+5q)^3 +3(2p+5q)^2(5q-2p)+3(2p+5q)(5q-2p)^2+ (5q-2p)^3$

ধরি,
$ 2p+5q = a $ এবং
$ 5q-2p= b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^3+3a^b+3ab^2+b^3$
=$(a+b)^3 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $(2p+5q +5q-2p)^3 $
= $(10q)^3 $
= $1000q^3 $

অতএব,
নির্ণেয় মান: = $1000q^3 $

৪.২.২(গ) সরল কর:
$(x+2y)^3 -3(x+2y)^2(x-2y)+3(x+2y)(x-2y)^2- (x-2y)^3$
সমাধান:
প্রদত্ত রাশী
= $(x+2y)^3 -3(x+2y)^2(x-2y)+3(x+2y)(x-2y)^2- (x-2y)^3$

ধরি,
$ x+2y = a $ এবং
$ x-2y= b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^3-3a^b+3ab^2-b^3$
=$(a-b)^3 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $ \{(x+2y)-(x-2y)\}^3 $
= $(x+2y-x+2y)^3 $
= $(4y)^3 $
= $64y^3 $

অতএব,
নির্ণেয় মান: = $64y^3 $

৪.২.২(ঘ) সরল কর:
$(6m+2)^3 -3(6m+2y)^2(6m-4)+3(6m+2)(6m-4)^2- (6m-4)^3$
সমাধান:
প্রদত্ত রাশী
= $(6m+2)^3 -3(6m+2y)^2(6m-4)+3(6m+2)(6m-4)^2- (6m-4)^3$

ধরি,
$ 6m+2 = a $ এবং
$ 6m-4= b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^3-3a^b+3ab^2-b^3$
=$(a-b)^3 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $ \{(6m+2)-(6m-4)\}^3 $
= $(6m+2-6m+4)^3 $
= $(6)^3 $
= $216 $

অতএব,
নির্ণেয় মান: = $216 $

৪.২.২(ঙ) সরল কর:
$(x-y)^3 + (x+y)^3 +6x(x^2-y^2)$
সমাধান:
প্রদত্ত রাশী
= $(x-y)^3 + (x+y)^3 +6x(x^2-y^2)$
= $(x-y)^3 + (x+y)^3 +6x(x+y)(x-y)$
= $(x-y)^3 + (x+y)^3 +6(x+y+x-y)(x+y)(x-y)$
সাজিয়ে লিখলে পাই,
= $(x-y)^3 + (x+y)^3 +6(x-y)(x+y) (x+y+x-y)$

ধরি,
$ x-y = a $ এবং
$ x+y= b $
সুতরাং, আমরা পাই,
প্রদত্ত রাশী = $a^3+b^3+3ab(a+b)$
=$(a+b)^3 $

এখন, মান বসিয়ে পাই,
প্রদত্ত রাশী = $ \{(x-y)+(x+y)\}^3 $
= $(x-y+x+y)^3 $
= $(2x)^3 $
= $8x^3 $

অতএব,
নির্ণেয় মান: = $8x^3 $
