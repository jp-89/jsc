২.২.১২। এক ব্যক্তি ঋণদান সংস্থা থেকে বার্ষিক ৮% চক্রবৃদ্ধি মুনাফায় ৫০০০ টাকা ঋণ নিলেন। প্রতিবছর শেষে তিনি ২০০০ টাকা করে পরিশোধ করেন। ২য় কিস্তি পরিশোধের পর তার আর কত টাকা ঋণ থাকবে?
সমাধান:
আমরা জানি চক্রবৃদ্ধি মূলধনের ক্ষেত্রে,
$ C = P(১+r)^n $

দেওয়া আছে, প্রথম কিস্তির ক্ষেত্রে,
প্রারম্ভিক মূলধন = P =৫০০০ টাকা
সুদের হার = $ r = ৮ \text" " % = {৮}/{১০০} $

সময় n = ১ বছর।
চক্রবৃদ্ধি মূলধন C = ?

মান বসিয়ে পাই,
$ C = ৫০০০ ×(১+{৮}/{১০০} )^১ $

= $ ৫০০০ ×({১০০+৮}/{১০০} )^১ $

= $ ৫০০০×({১০৮}/{১০০} ) $

= $ ৫০ ×{১০৮} $

= $৫৪০০ $ টাকা

প্রথম কিস্তির শেষে টাকা পরিশোধ করে ২০০০ টাকা।

অতএব,
প্রথম কিস্তি পরিশোধ শেষে ঋণ থাকে = $৫৪০০ - ২০০০ $টাকা।
= ৩৪০০ টাকা।

দ্বিতীয় কিস্তির ক্ষেত্রে,
প্রারম্ভিক মূলধন = P =৩৪০০ টাকা
সুদের হার = $ r = ৮ \text" " % = {৮}/{১০০} $

সময় n = ১ বছর।
চক্রবৃদ্ধি মূলধন C = ?

মান বসিয়ে পাই,
$ C = ৩৪০০ ×(১+{৮}/{১০০} )^১ $

= $ ৩৪০০ ×({১০০+৮}/{১০০} )^১ $

= $ ৩৪০০×({১০৮}/{১০০} ) $

= $ ৩৪ ×{১০৮} $

= $৩৬৭২ $ টাকা

দ্বিতীয় কিস্তির শেষে টাকা পরিশোধ করে ২০০০ টাকা।

অতএব,
দ্বিতীয় কিস্তি পরিশোধ শেষে ঋণ থাকে = $৩৬৭২ - ২০০০ $ টাকা।
= ১৬৭২ টাকা।

অতএব,
দ্বিতীয় কিস্তি পরিশোধ শেষে তার ঋণ থাকবে ১৬৭২ টাকা।
