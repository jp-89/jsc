২.২.১১। কোনো শহরের বর্তমান জনসংখ্যা ৬৪ লক্ষ। শহরটির জনসংখ্যা বৃদ্ধির হার প্রতি হাজারে ২৫ জন হলে, ২ বছর পর ঐ শহরের জনসংখ্যা কত হবে?
সমাধান:
দেওয়া আছে,
শহরের বর্তমান জনসংখ্যা = ৬৪ লক্ষ।
= ৬৪০০০০০

জনসংখ্যা বৃদ্ধির হার প্রতি হাজারে ২৫ জন।
শতকরা বৃদ্ধির হার = $ {২৫ × ১০০}/{১০০০ } $

= ২.৫ %

সময় n = ২ বছর।


আমরা জানি চক্রবৃদ্ধির ক্ষেত্রে,
$ C = P(১+r)^n $

এখানে, P = ৬৪০০০০০
r = ২.৫ % = $ { ২.৫}/{ ১০০} $

n = ২ C = ২ বছর পর ঐ শহরের জনসংখ্যা।
মান বসিয়ে পাই,
$ C = ৬৪০০০০০ ×(১+{২.৫}/{১০০} )^২ $

= $ ৬৪০০০০০ ×({১০০+২.৫}/{১০০} )^২ $

= $ ৬৪০০০০০×({১০২.৫}/{১০০} )^২ $

= $ ৬৪০০০০০ ×{১০২.৫}/{১০০} ×{১০২.৫}/{১০০} $

= $ ৬৪০×১০২.৫$ ×১০২.৫

= $৬৭২৪০০০ $

অতএব,
ঐ শহরের জনসংখ্যা = $৬৭২৪০০০ $ জন।
