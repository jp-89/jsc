২.২.৮। বার্ষিক শতকরা ১০ টাকা মুনাফায় ৫০০০ টাকার ৩ বছরের সরল মুনাফা ও চক্রবৃদ্ধি মুনাফার পার্থক্য কত হবে?
সমাধান:
আমরা জানি, সরল মুনাফার ক্ষেত্রে,
I = Prn
অর্থাৎ, $ \text " মুনাফা " ={ \text" আসল " × { \text " মুনাফার হার " } × \text" সময় " } $

এখানে,
মুনাফা = I = ?
আসল = P = ৫০০০ টাকা।
মুনাফার হার = r = ১০% = ${১০}/{১০০ }$
সময় = n = ৩ বছর।

অতএব,
মুনাফা = $৫০০০ × {১০}/{১০০ } ×৩ $

= ১৫০০ টাকা।

আবার , আমরা জানি চক্রবৃদ্ধি মূলধনের ক্ষেত্রে,
$ C = P(১+r)^n $

দেওয়া আছে,
প্রারম্ভিক মূলধন P = ৫০০০ টাকা।
সুদের হার $ r = ১০% $
= $ {১০}/{১০০}$

=$.১$
সময় n = ৩ বছর।
চক্রবৃদ্ধি মূলধন = ?

মান বসিয়ে পাই,
$ C = ৫০০০ ×(১+.১)^৩ $
= $ ৫০০০×(১.১)^৩ $
= $ ৫০০০×({১১}/{১০ })^৩ $

= $ ৫০০০×{১৩৩১} /{১০০০}$

= $৬৬৫৫$ টাকা।

অতএব,
চক্রবৃদ্ধি মূলধন C = $ ৬৬৫৫ $ টাকা।

চক্রবৃদ্ধি মুনাফা = C-P
= $ ৬৬৫৫ - ৫০০০ $
= $ ১৬৫৫ $

চক্রবৃদ্ধি ও সরল মুনাফা মুনাফার পার্থক্য = চক্রবৃদ্ধি মুনাফা - সরল মুনাফা
$ ১৬৫৫ - ১৫০০ $ টাকা।
= $ ১৫৫ $ টাকা।
