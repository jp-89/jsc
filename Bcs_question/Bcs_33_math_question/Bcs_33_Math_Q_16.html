১৬ । নিচের কোনটি $(√5 - √3) $ এর সমান?
Ο ক) $ √2 $

Ο খ) $1/{2(√5 - √3) } $

Ο গ) $ 1/{√5 + 1/√3 }$

Ο ঘ) $2/{(√5 + √3)}$

সঠিক উত্তর: (ঘ)
সমাধান:
দেওয়া আছে,
$(√5 - √3) $

= ${(√5 - √3)(√5 + √3)}/{ (√5 + √3) } $

= ${(√5)^2 - (√3)^2}/{ (√5 + √3) } $

= ${5 - 3}/{ (√5 + √3) } $

= ${2}/{ (√5 + √3) } $
