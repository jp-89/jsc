৫.৩.৬। সমাধান কর:
 ${ 3^{3x-4} . a^{2x-5} }/{ 3^{x+1} } = a^{2x-5} $ (a> 0)

 সমাধান:
প্রদত্ত রাশি = ${ 3^{3x-4} . a^{2x-5} }/{ 3^{x+1} } = a^{2x-5}$

বা, ${ 3^{3x-4} }/{ 3^{x+1} } = {a^{2x-5}}/{ a^{2x-5}} $

বা, ${ 3^{3x-4} }/{ 3^{x+1} } = 1 $

বা, ${ 3^{3x-4} } = { 3^{x+1} } $

অতএব,
$ 3x-4 = x+1 $

বা, $ 3x-x = 1+4$

বা, $2x= 5$

বা, ${x}= {5/2}$

নির্ণেয় $x = 5/2$
