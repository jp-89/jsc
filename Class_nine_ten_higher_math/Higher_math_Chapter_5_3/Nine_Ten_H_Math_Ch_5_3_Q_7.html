৫.৩.৭। সমাধান কর:
 $ {5^{3x-5} . b^{2x-6} }/{ 5^{x+1} } = a^{2x-6}$ , ( a> 0, b> 0, 5b ≠ a)

সমাধান:
প্রদত্ত রাশি = $ {5^{3x-5} . b^{2x-6} }/{ 5^{x+1} } = a^{2x-6} $

বা, $ {5^{3x-5-x-1} . b^{2x-6} } = a^{2x-6} $

বা, $ {5^{2x-6} . b^{2x-6} } = a^{2x-6} $

বা, $ {(5b)^{2x-6} } = a^{2x-6} $

বা, $ {(5b)^{2x-6} }/{ a^{2x-6} } = 1 $

বা, $ {({5b}/a)^{2x-6} } = 1 $

বা, $ {({5b}/a)^{2x-6} } = ({5b}/a)^0 $

অতএব,
$ 2x-6 = 0 $

বা, $ 2x = 6$

বা, ${x}= {6/2}$

বা, ${x}= {3}$

নির্ণেয় $x = 3$
