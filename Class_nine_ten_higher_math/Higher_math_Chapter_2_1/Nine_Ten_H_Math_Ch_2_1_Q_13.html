২.১৩। আংশিক ভগ্নাংশে পরিণত কর:
 (a) ${5x+4 }/{ x(x+2)}$

 সমাধান:
প্রদত্ত রাশি = ${5x+4 }/{ x(x+2)}$

ধরি,
${5x+4 }/{ x(x+2)} = A/x + B/{x+2}$ ....(1)

উভয় পক্ষকে $ x(x+2)$ দ্বারা গুন্ করিয়া পাই,
বা, $5x+4 = A(x+2)+ Bx$ ....(2)

(2) সমীকরণে $ x= -2 $ বসিয়ে পাই,
$5(-2)+4 = A(-2+2)+ B(-2)$

বা, $-10+4 = A(0)- 2B$

বা, $-6 = - 2B$

বা, $2B = 6$

বা, $B = 3$

আবার, (2) সমীকরণে $ x= 0 $ বসিয়ে পাই,
$5(0)+4 = A(-0+2)+ B(0)$

বা, $ 4 = 2A$

বা, $2A = 4$

বা, $A = 2$

(1) সমীকরণে A এবং B এর মান বসিয়ে পাই ,
${5x+4 }/{ x(x+2)} = 2/x + 3/{x+2}$

নির্ণেয় আংশিক ভগ্নাংশ= $2/x + 3/{x+2}$

২.১৩। আংশিক ভগ্নাংশে পরিণত কর:
 (b) ${x+2 }/{ x^2-7x+12}$

 সমাধান:
প্রদত্ত রাশি = ${x+2 }/{ x^2-7x+12}$

= ${x+2 }/{ x^2-4x-3x+12}$

= ${x+2 }/{ x(x-4)-3(x-4}$

= ${x+2 }/{ (x-3)(x-4)}$

ধরি,
${x+2 }/{ (x-3)(x-4)}= A/{x-3} + B/{x-4}$ ....(1)

উভয় পক্ষকে $(x-3)(x-4)$ দ্বারা গুন্ করিয়া পাই,
বা, $x+2 = A(x-4)+ B(x-3)$ ....(2)

(2) সমীকরণে $ x= 4 $ বসিয়ে পাই,
$4+2 = A(4-4)+ B(4-3)$

বা, $6 = A(0)+ B$

বা, $6 = B$

বা, $B = 6$

বা, $B = 6$

আবার, (2) সমীকরণে $ x= 3 $ বসিয়ে পাই,
$3+2 = A(3-4)+ B(3-3)$

বা, $ 5 = -A+B(0)$

বা, $-A = 5$

বা, $A = -5$

(1) সমীকরণে A এবং B এর মান বসিয়ে পাই,
${ x+2 }/{ (x-3)(x-4)}= {-5}/{x-3} + 6/{x-4}$

=$ 6/{x-4} - 5/{x-3}$

নির্ণেয় আংশিক ভগ্নাংশ= $ 6/{x-4} - 5/{x-3}$

২.১৩। আংশিক ভগ্নাংশে পরিণত কর:
 (c) ${x^2-9x -6}/{ x(x-2)(x+3)}$

 সমাধান:

বা, $x^2-9x -6 = A(x-2)(x+3)+ Bx(x+3)+ Cx(x-2)$ ....(2)

(2) সমীকরণে $ x= 0 $ বসিয়ে পাই,

$ -6 = -6A+ B(0)+ C(0)$

বা, $6A = 6$

বা, $A = 1$

আবার, (2) সমীকরণে $ x= 2 $ বসিয়ে পাই,

বা, $4-18 -6 = A(0)(5)+ B(2)(5)+ C(2)(0)$

বা, $-20 = 0+ 10B+ 0$

বা, $9+27-6 = A(-5)(0)+ B(-3)(0)+ C(-3)(-5)$
