(a) $x $ এর মান
(b) $n(A ∪B ) $ এবং $n(A ∩B^' ) $

সমাধান:
(a) $x $ এর মান

এখানে,
n(A) হল A সেটের উপাদান সংখ্যা।
n(B) হল B সেটের উপাদান সংখ্যা।
অতএব,
$n(A) = 3x+x$
$n(B) = 2x+8+x$

শর্তানুসারে,
$n(A) = n(B)$
বা, $ 3x+x = 2x+8+x$
বা, $ 3x+x-2x-x = 8$
বা, $ x = 8$

(b) $n(A ∪B ) $ এবং $n(A ∩B^' ) $
আমরা জানি,
$n(A ∪B ) = n(A)+ n(B) - n(A ∩B)$
এখানে,
n(A ∩B) হল A এবং B সেটের সাধারণ উপাদান সংখ্যা।
অতএব,
$ n(A ∩B) = x $
মান বসিয়ে পাই,
$n(A ∪B ) = 3x+x+ 2x+8+x - x$
= $6x+8$

$x=8 $ মান বসিয়ে পাই,
$n(A ∪B ) = 6×8+8$
= $48+8$
=$56$

এবং
$n(A ∩B^' ) = n(A) - n(A ∩B)$
মান বসিয়ে পাই,
=3x+x-x
=3x
=$3×8$
=$24$
