সমাধান:
(a) যদি $n(U) = 50 $ হয়, তবে x এর মান নির্ণয় কর।

ভেনচিত্র থেকে পাই,
$n(U) = 2x + x+1 + x-1 +2+3+0+x+5$
=$5x+10$

শর্তানুসারে,
$5x+10=50$
বা, $5x=50-10$
বা, $5x=40$
বা, $x={40}/5$
বা, $x=8$

(b) $n (B ∩ C^' )$ এবং $ n (A^' ∩ B )$ এর মান নির্ণয় কর।

$n(B ∩C^' ) = n(B) - n(B ∩C)$
সাধারণ কথায় B থেকে B এবং C এর সাধারণ উপাদান সমূহ বাদ
মান বসিয়ে পাই,
=$(x+1+x-1+3)-3$
=$2x$
=$2×8$
=$16$

এখন,
$ n (A^' ∩ B )= n (B∩ A^' )$
=$ n(B) - n(B ∩A)$
সাধারণ কথায় B থেকে A এবং B এর সাধারণ উপাদান সমূহ বাদ।
=$ (x+1+x-1+3) - (x+1+3$
=$x-1$
=$8-1$
=$7$

(c ) n (A ∩ B ∩^' C ) এর মান নির্ণয় কর।
$n (A ∩ B ∩ C^' ) = n (A ∩ B ∩ C^' )$
সাধারণ কথায় A B এর সাধারণ উপাদানথেকে A,B এবং C এর সাধারণ উপাদান সমূহ বাদ
মান বসিয়ে পাই,
=$(x+1+3)-3$
=$x+1
