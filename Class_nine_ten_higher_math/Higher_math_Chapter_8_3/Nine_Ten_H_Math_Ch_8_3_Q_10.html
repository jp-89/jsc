৮.৩.১০। $θ= π/3 $ হলে নিম্নোক্ত অভেদসমূহ যাচাই কর:
(i) $ sin2θ = 2sinθ cosθ = {2tanθ}/{1+tan^2 θ}$
সমাধান:
দেওয়া আছে,
প্রথম পক্ষ = $sin2θ$
= $ sin{2π}/3 $
= $ sin(π -π/3) $
= $ sin π/3 $

= ${√3}/2$

অতএব,
প্রথম পক্ষ = ${√3}/2$

দ্বিতীয় পক্ষ = $2sinθ cosθ$
= $ 2sin π/3 cosπ/3 $

= $2×{√3}/2 ×1/2$

= ${√3}/2$

অতএব,
দ্বিতীয় পক্ষ = ${√3}/2$

তৃতীয় = ${2tanθ}/{1+tan^2 θ}$
= $ {2tan π/3}/{1+tan^2 π/3} $

= ${2×√3}/{1+ (√3)^2}$

= ${2×√3}/{1+ 3}$

= ${2×√3}/{4}$

= ${√3}/2$

অতএব,
তৃতীয় পক্ষ = ${√3}/2$

অতএব,
প্রথম পক্ষ = দ্বিতীয় পক্ষ = তৃতীয় পক্ষ (প্রমাণিত)

৮.৩.১০। $θ= π/3 $ হলে নিম্নোক্ত অভেদসমূহ যাচাই কর:
(ii) $ sin3θ = 3sinθ -4sin^3 θ $
সমাধান:
দেওয়া আছে,
বাম পক্ষ = $sin3θ$
= $ sin{3π}/3 $
= $ sin π $
= $0$

অতএব,
বাম পক্ষ = $0$

ডান পক্ষ = $3sinθ -4sin^3 θ$
= $ 3sin π/3 -4sin^3 π/3 $

= $3 ×{√3}/2 - 4×({√3}/2)^3$

= ${3√3}/2 - {4×3√3}/8$

= ${3√3}/2 - {{3√3}/2$

= $0$

অতএব,
ডান পক্ষ = $0$

অতএব,
বাম পক্ষ = ডান পক্ষ (প্রমাণিত)

৮.৩.১০। $θ= π/3 $ হলে নিম্নোক্ত অভেদসমূহ যাচাই কর:
(iii) $ cos3θ = 4cos^3θ -3cos θ $
সমাধান:
দেওয়া আছে,
বাম পক্ষ = $cos3θ$
= $ cos{3π}/3 $
= $ cos π $
= $-1$

অতএব,
বাম পক্ষ = $-1$

ডান পক্ষ = $4cos^3 θ -3cos θ$
= $ 4cos^3 π/3 -3cos π/3 $

= $4 ×({1}/2)^3 - 3×1/2$

= $4 ×1/8 - 3×1/2$

= $1/2 - 3/2$

= ${1-3}/2$

= ${-2}/2$

= $-1$

অতএব,
ডান পক্ষ = $-1$

অতএব,
বাম পক্ষ = ডান পক্ষ (প্রমাণিত)

৮.৩.১০। $θ= π/3 $ হলে নিম্নোক্ত অভেদসমূহ যাচাই কর:
(iv) $ tan2θ = {2tanθ}/{1-tan^2 θ} $
সমাধান:
দেওয়া আছে,
বাম পক্ষ = $tan2θ$
= $ tan{2π}/3 $
= $ tan (π-π/3) $
= $ -tan π/3 $
= $-√3$

অতএব,
বাম পক্ষ = $-√3$

ডান পক্ষ = ${2tanθ}/{1-tan^2 θ}$
= $ {2tan π/3}/{1-tan^2 π/3} $

= ${2 ×√3} /{ 1-(√3)^2 }$

= ${2√3} /{ 1-3 }$

= ${2√3} /{ -2 }$

= $-√3$

অতএব,
ডান পক্ষ = $-√3$

অতএব,
বাম পক্ষ = ডান পক্ষ (প্রমাণিত)
